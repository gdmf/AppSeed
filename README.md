# AppSeed ES6

## Prerequisites
- node.js > 4x
- bower & yarn ($ npm i -g yarn bower)


## What's in it?
- Bower for frontend dependencies
- Expressjs for the server
- ES6 Modules
- Webpack (CSS/JS)
- Bable (Transpiling)
- ESLint
- Testing with Mocha
- Test Reports MochaAwesome
- nsp (node security project)
- simple frontend router


## Download Packages
~~~
  $ npm run up
~~~


## Start Developing
~~~
  $ npm start -s
~~~


## Build Distribution
~~~
  $ npm run build -s
~~~
